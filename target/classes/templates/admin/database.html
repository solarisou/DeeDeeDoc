Actions Rapides

Réinitialisation Complète (RECOMMANDÉ)

Fait TOUT en un clic : Supprime les tables + Recrée le schéma + Charge les données + Supprime les fichiers

C'est le bouton à utiliser après avoir appuyé sur "Supprimer toutes les tables"
Ou actions individuelles :
Créer le schéma

Crée les tables depuis database_schema.sql

mysql < database_schema.sql
Charger données

Importe sample_data.sql (tables doivent exister)

mysql < sample_data.sql
Vider uploads

Supprime tous les PDF dans uploads/

Fichiers : 0

Supprimer Toutes les Tables (DROP)

DROP supprime complètement les tables de la base de données. Vous devrez recréer le schéma avec le fichier database_schema.sql.

Supprimer toutes les tables

Cette action détruit complètement toutes les tables de la base de données.
Utilisez ensuite "Réinitialisation Complète" pour recréer la base.
